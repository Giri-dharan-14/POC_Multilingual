# 🗣️ Multilingual Code-Mixed Voice AI System

A comprehensive proof-of-concept for handling code-mixed South Indian languages using OpenAI's GPT-4o-mini, Whisper, and TTS models. This system can understand, detect, and respond to Tamil, Telugu, Kannada, and Malayalam mixed with English - just like how people naturally speak in urban South India!

## ✨ Features

### 🎯 Core Capabilities
- **Language Detection**: Automatic detection of code-mixed South Indian languages
- **Speech-to-Text**: Audio transcription with language identification using Whisper
- **Text-to-Speech**: Natural code-mixed speech generation
- **Real-time Chat**: Interactive voice conversations with LiveKit integration
- **Cultural Context**: Region-specific responses with appropriate greetings and cultural references

### 🌏 Supported Languages
- **Tamil** (தமிழ்) - Tamil Nadu culture and Chennai references
- **Telugu** (తెలుగు) - Andhra/Telangana culture and Hyderabad references  
- **Kannada** (ಕನ್ನಡ) - Karnataka culture and Bangalore references
- **Malayalam** (മലയാളം) - Kerala culture and backwater references
- **Code-Mixed** combinations with English
